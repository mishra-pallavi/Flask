create REST APIs with Flask-RESTX

Steps

step1 : acivate virtual enviroment

   python3 -m venv env
   
   source env/bin/activate
   
step2 : install flask

   pip install flask
   
step3 : install restx 

   pip install flask-restx
   
step4 : install sqlalchemy

   pip install flask-sqlalchemy
   
step5 : run project

   flask run
